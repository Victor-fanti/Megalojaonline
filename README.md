Protótipo de Click & Collect - MegaLoja Online
Este projeto consiste em um protótipo de baixa fidelidade para um sistema de Click & Collect (Compre e Retire), desenvolvido com o objetivo de validar fluxos de interface e experiência do usuário (UX) tanto para clientes quanto para funcionários.

📋 Descrição do Projeto
O sistema permite que clientes realizem compras online e selecionem uma loja física próxima para retirada, agendando o melhor horário. Simultaneamente, fornece um dashboard para funcionários gerenciarem esses pedidos, desde a separação dos itens até a confirmação da entrega final.

🚀 Funcionalidades
Para o Cliente
Seleção de Loja: Escolha de unidades próximas com base em disponibilidade e distância.

Agendamento: Seleção de horários disponíveis para retirada.

Confirmação: Geração de um QR Code para identificação rápida na loja.

Rastreamento: Acompanhamento do status do pedido (Preparando, Pronto, Retirado).

Para o Funcionário
Dashboard em Tempo Real: Visão geral de pedidos novos, em preparação e prontos.

Gestão de Pedidos: Fluxo simplificado para marcar pedidos como "Prontos" ou "Retirados".

Histórico: Acesso ao status de todos os pedidos processados.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica das páginas.

CSS3: Estilização responsiva, utilizando Grid e Flexbox para layout, além de variáveis de design para consistência.

JavaScript (Vanilla): Lógica de estado da aplicação, manipulação do DOM para navegação entre telas e simulação de fluxos.

🎨 Decisões de Design
Paleta de Cores: Focada em transmitir confiança (Azul #1F4788) e ação (Turquesa #00A3E0).

Tipografia: Uso de Arial para garantir alta legibilidade em diferentes dispositivos.

Acessibilidade: Botões projetados com altura mínima de 48px e contraste de cores conforme diretrizes de acessibilidade.

Responsividade: Layout fluido que se adapta de dispositivos mobile (375px) a telas de desktop.

📂 Como Utilizar
Clone este repositório ou baixe os arquivos.

Certifique-se de que os arquivos index.html e index.css estejam na mesma pasta (embora o estilo esteja embutido no HTML, manter a estrutura original é recomendado).

Abra o arquivo index.html em qualquer navegador web moderno para interagir com os protótipos.
