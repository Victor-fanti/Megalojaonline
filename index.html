📋 Notas de Design

Decisões de Design:

  • Paleta de cores: Azul (#1F4788) para confiança + Turquesa (#00A3E0) para ação
  • Tipografia: Arial para máxima legibilidade
  • Espaçamento: 20px padding padrão para conforto visual
  • Ícones: Emojis para rápida identificação em baixa fidelidade
  • Feedback: Estados de hover e active bem definidos
  • Responsividade: Ajustável para mobile (375px) e desktop (1200px+)
  • Acessibilidade: Contraste WCAA AA mínimo, botões >= 48px altura

Fluxos Implementados:

  • Cliente: Home → Selecionar Loja → Agendar → Confirmação → Rastreamento
  • Funcionário: Dashboard → Detalhe Pedido → Separação → QR Validation → Entrega

Próximos Passos:

  • Prototipagem em alta fidelidade no Figma
  • Teste de usabilidade com 5 usuários por persona
  • Refinamento baseado em feedback
  • Especificação de componentes e design system
